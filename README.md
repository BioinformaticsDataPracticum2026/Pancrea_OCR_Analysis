### Pancrea_OCR_Analysis
##### This project aims to answer the following questions:
    1. Which human and mouse ATAC-seq datasets are usable?
    2. Which open chromatin regions are shared or species-specific?
    3. What genes / biological processes are linked to those regions?
    4. Are those regions more enhancer-like or promoter-like?
    5. Which TFs may regulate them?
